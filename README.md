## 📌 Sobre o projeto

Portfólio pessoal desenvolvido do zero durante minha transição de carreira para a área de Tecnologia. O objetivo foi praticar HTML e CSS na prática, criando algo real e funcional para apresentar meu perfil profissional.

---

## ✨ Funcionalidades

- Cards com efeito flip 3D ao passar o mouse
- Barras de habilidades animadas com scroll
- Ticker animado com informações em destaque
- Seção de experiência em formato de timeline
- Layout responsivo para mobile e desktop
- Animações de entrada com Intersection Observer
